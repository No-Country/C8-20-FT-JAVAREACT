<h2>Descripción</h2>
<p>Villain Burger es una tienda de comida desarrollada con React Js en el front-end css personalizado en los estilos y Spring boot , Spring security , Jwt en el back-end, tambien es responsive por lo que se adapta a los direntes tipos y tamaños de pantallas. Cuenta con capacidad de tomar pedidos como clientes agrandando productos a un carrito de compras. Tambien posee un dashboard-admin en el cual podemos editar, agregar, actualizar o borrar algunas opciones o caracteristicas.</p>
